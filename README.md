# Web_Page

動態網頁程式設計

***

* php 檔案執行方法

```
http://localhost/你的檔案.php
```

## php 常用語法

# 變數宣告

在宣告變數時 不用先表明他的資料型態 只要在變數名稱前加上`$`就行了

```
$var = 1
```

# echo 與 print 的差別

print 只能輸出單一字串

echo 能輸出很多個字串

# 換行

```
echo "換行<br/>"
```

# 標題標籤

```
<?php
    echo "網頁程式設計";
    echo "<h1>網頁程式設計</h1>;
    echo "<h2>網頁程式設計</h2>;
    echo "<h3>網頁程式設計</h3>;
    echo "<h4>網頁程式設計</h4>;
    echo "<h5>網頁程式設計</h5>;
    echo "<h6>網頁程式設計</h6>;
?>
```

# 分隔線

根據網頁尺寸滾動式調整

```
echo "<hr>";
```

# 粗體 & 斜體

```
echo "<b>這是粗體</b>";
echo "<br/>"
echo "<i>這是斜體</i>";
```
